# cupcake-muffin-classifier
Cupcake vs Muffin Classifier
A binary classification model that predicts whether a recipe is a cupcake or a muffin based on flour and sugar content.
Algorithm: Support Vector Machine (SVM) with a linear kernel
Dataset: recipes_muffins_cupcakes.csv
Features used: Flour, Sugar
Accuracy: 100%
Libraries: NumPy, Pandas, Scikit-learn, Matplotlib, Seaborn
How it works: The SVM finds the optimal hyperplane separating muffin and cupcake recipes based on two ingredients. The decision boundary and support vectors are visualized using Matplotlib.
